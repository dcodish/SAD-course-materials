# תרגיל בית — חקר מצב קיים: מוסך "גלגל חמישי" 🔧

> תרגול עצמי אחרי הרצאה 2 · **לא להגשה** — אבל זו בדיוק העבודה שתעשו בפרויקט ובמבחן.
> מומלץ לעבוד בקבוצת הפרויקט שלכם. זמן משוער: 60–90 דקות.

אספתם חומר גלם על תהליך הטיפול ברכב במוסך "גלגל חמישי". כמו בשטח האמיתי — **דיברתם
עם כמה אנשים, כל אחד מזווית שלו**, וכל אחד סיפר את החלק שהוא מכיר, לא בהכרח לפי הסדר,
עם דעות והרגלים באמצע. אף אחד לא סתר את השני בלוגיקה — אבל כדי לצייר את התהליך
תצטרכו **להצליב את הקולות ולסדר אותם לרצף אחד**.

המשימה: לפענח מהחומר את התהליך הנכון, ואז **לבנות ממנו תרשים BPMN בעזרת AI** — בדיוק
כמו שראינו בכיתה (טקסט → תרשים).

## 🎯 מה עליכם להפיק

1. **רשימת המשתתפים** — מי בריכה נפרדת ומי נתיב? מי בריכה **פתוחה** ומי **ריקה
   (black box)**? נמקו לפי הכלל מהכיתה: *מי מנהל את התהליך?*
2. **רצף התהליך במילים** — סדרו את הצעדים לפי הסדר הכרונולוגי הנכון, כולל ההסתעפויות
   (מתי כן / מתי לא) ונקודת החזרה (לולאה) אם יש.
3. **תרשים BPMN** — בנו אותו **בעזרת AI**: כתבו תיאור מסודר של התהליך, בקשו מ-Claude
   לייצר BPMN, ובדקו את התוצאה מול הכללים. דרישות מינימום כמו בפרויקט: בריכה עם ≥2
   נתיבים, ≥8 פעילויות, ≥2 שערים, וזרימות מסר לבריכות החיצוניות.

> 💡 שימו לב במיוחד ל: **שער האישור** של הלקוח, מה קורה כש**חלק חסר במלאי**, מה קורה
> אם **בדיקת הדרך נכשלת**, ואילו משתתפים הם **קופסה שחורה**.
