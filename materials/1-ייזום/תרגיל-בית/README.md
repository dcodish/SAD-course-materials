# תרגיל בית — ייזום: "פיצה נונה" 🍕

> תרגול עצמי אחרי הרצאה 1 · **לא להגשה** — אבל זו בדיוק העבודה שתעשו בפרויקט ובמבחן.
> מומלץ לעבוד בקבוצת הפרויקט שלכם. זמן משוער: 60–90 דקות.

ג'קי, הבעלים של פיצריית "פיצה נונה" בבאר שבע, פנה אליכם: המשלוחים שלו מאחרים,
הלקוחות מתלוננים, והוא כבר יודע (כך הוא אומר) מה הבעיה. תפקידכם — כמו שעשינו
בכיתה עם הקפיטריה של שמעון — **לא להאמין לו על המילה**, אלא לנתח את חומר הגלם
ולהגיע בעצמכם לשורש.

## 🎯 מה עליכם להפיק

כמו בהרצאה — קודם הבעיה (חלקים 1–4), ורק אחר כך פתרון:

1. **ניסוח בעיה מלא** לפי תבנית 7 החלקים — שימו לב: מצב אידיאלי מתאר **חוויה**, לא פתרון.
2. **Fishbone** — בחרו בעצמכם את הקטגוריות שמתאימות לבעיה הזו.
3. **שרשרת 5 Whys** — מהסימפטום ועד שורש שאפשר לפעול עליו.
4. **השלכות כלכליות מכומתות** — מהנתונים שבחומר, לא מהרגשה.
5. **שורש מומלץ + הראיות** — כל טענה מעוגנת בציטוט או בנתון מהחומר.

בונוס: עץ FTA לבעיה, ו"שאלת הראיון הבאה" — מה הייתם שואלים ואת מי כדי לאמת את השורש.
