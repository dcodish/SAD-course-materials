איך משתמשים: לחצו על הרצאה לצפייה בדפדפן. ניווט: חצים / רווח = קדימה ואחורה, F = מסך מלא, O = מפת שקופיות. להורדה למחשב — הורידו את כל המאגר כ-ZIP (כפתור Code → Download ZIP) ופתחו את index.html של ההרצאה.

כל הקבצים וחומרי העזר (קובצי Markdown, דוגמאות ותרגילים) נמצאים במאגר ה-GitHub של הקורס: github.com/dcodish/SAD-course-materials

🔌 ה-MCP של הקורס: הורידו את sad-mcp.mcpb, פתחו ב-Claude Desktop את Settings → Extensions וגררו את הקובץ לתוך העמוד — זהו, אתם מחוברים (בשימוש הראשון תתבקשו להזדהות עם חשבון @post.bgu.ac.il).
